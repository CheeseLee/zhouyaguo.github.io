搭建
----

- mkdir /pypi
- pip install pip2pi
- for i in `find . -type f -name "requirements\.txt" `; do pip2tgz /pypi -r $i; echo "pip2tgz $i ..."; done
- dir2pi /pypi  #建立索引

使用
---

```
vim ~/.pip/.pip.conf
[global]
index-url = http://your_host_ip/pypi/simple
```
